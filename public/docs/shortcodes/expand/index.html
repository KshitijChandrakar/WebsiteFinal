    Expand #

    Expand shortcode can help to decrease clutter on screen by hiding part of text. Expand content by clicking on it.

    Example #

    Default # 

    {{< expand >}}
## Markdown content
Lorem markdownum insigne...
{{< /expand >}}

    With Custom Label # 

    {{< expand "Custom Label" "..." >}}
## Markdown content
Lorem markdownum insigne...
{{< /expand >}}
